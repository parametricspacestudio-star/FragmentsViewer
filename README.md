# Fragments Viewer

This is a lightweight viewer [Fragments (.frag)](https://github.com/ThatOpen/engine_fragment) files. The example model is provided by [Enzyme APD](https://www.weareenzyme.com), and was exported with the [Archicad Fragments Exporter](https://github.com/bimdots-dev/FragmentsArchicadAddOn).

## Drag & Drop

Simply drag and drop a .frag file into the viewer window to load it. The viewer supports both compressed and uncompressed .frag files.

## Define model location in the URL

If your .frag file is hosted on a web server, you can load it directly by specifying the file's location in the viewer URL. Use the URL scheme below to create such a link.
```
https://bimdots-dev.github.io/FragmentsViewer/#<FragFileURL>
```

Refer to [this example](https://bimdots-dev.github.io/FragmentsViewer/#https://dl.dropbox.com/scl/fi/3it3nyaj3xfko66yrnhky/stacked_towers.frag?rlkey=3y5n4t92ecqlm0ndq94xgb3on&st=xjs9zi7h) to load a file hosted on Dropbox.
